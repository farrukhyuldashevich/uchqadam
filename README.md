# UchQadam

A 60-day, single-store hyperlocal grocery delivery validation experiment in Tashkent (July–August 2026).

The goal is not to build quick commerce. The goal is to answer one question:
**Given a frictionless digital ordering channel, will people in this neighborhood order from this store and pay for delivery?**

## Repo contents

- [`docs/PRD.md`](docs/PRD.md) — the working product requirements document (timeline, success criteria, architecture, risks).
- `bot/` *(coming Week 3)* — Telegram bot source.
- `scripts/` *(coming Week 1)* — Google Sheets helpers and operational scripts.

## Stack (planned)

- Telegram bot via aiogram (Python) — *or* grammY (Node.js)
- Google Sheets as the catalog & order DB
- Free hosting tier (Railway / Render / Fly.io)
- Cash on delivery; Click/Payme integrated only if warranted
